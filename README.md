## 🇬🇧 English

### 📖 About the Project
AgentCraft AI is an autonomous Multi-Agent Swarm orchestration platform where specialized AI agents (Researcher, Analyst, Writer) collaborate to solve complex goals.

### ✨ Key Features
- 🤖 Autonomous Multi-Agent Swarm (Dr. Vance, Elena Rostova, Marcus Chen)
- 💬 Real-time inter-agent communication & reasoning log
- 🎯 Role-based task delegation & risk audit
- 📑 Automated executive Markdown report generation
- 🛡️ Enterprise security (Input sanitization & safe report export)

### 💻 Getting Started
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

## 🇫🇷 Français

### 📖 À propos du projet
AgentCraft AI est une plateforme d'orchestration de Multi-Agents IA autonomes où des agents spécialisés (Chercheur, Analyste, Rédacteur) collaborent pour résoudre des objectifs complexes.

- 💬 Communication inter-agents & journal de réflexion en direct

- 📑 Génération automatique de rapports exécutifs au format Markdown
- 🛡️ Normes de sécurité (Assainissement des entrées & exportation sécurisée)
